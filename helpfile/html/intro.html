An Interactive FPGA Circuit Simulator

 

Delta09 Circuit Simulator provides first year computer scientists with an easy way of prototyping digital circuits on an Altera DE2 FPGA board without using a hardware description language. It consists of a graphical circuit entry system allowing the user to select components from a library (including, 2- and 3-input NOR and NAND gates, RS-latch and D flip-flop, RAMs and ROMs) and place them on the screen. Switches and LEDs on the DE2 board are also available in the library to allow circuits to be interfaced to them. On-screen circuits can be saved to disk and loaded later. Circuits can also be written out as Verilog and then synthesised for direct implementation on the FPGA if a high performance implementation is desirable.
