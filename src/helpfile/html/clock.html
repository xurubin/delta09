Clocked Components

 

Delta Circuit Simulator has a single clock that controls all clocked components within the circuit you define. Clocked components can be recognised by the small inverted triangle at the bottom of the component, for example the “D flip-flop”.

 

The clock rate can be set using the control in the bottom left of the screen. The number is not measured in Hertz; it is in arbitrary units that vary according to the simulation speed, which is not user settable.
