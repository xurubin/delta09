Getting Started

 

After starting the circuit simulator, you will see the following window on screen:

The available components can be selected from the left-side collapsible menu and dragged
into the circuit-drawing area. Each component has some input connectors (indicated as red
nodes) and output connectors (indicated as grey nodes). By dragging the mouse between
desired nodes a wire will be created and connects the two target nodes. The wire routings
could be manually changed by first clicking the wire and then dragging the control points
(solid squares) attached to the wire. If extra control points are desirable, simply double click
the appropriate segment of the wire. The simulation clock frequency can be set using the
clock panel at the bottom left of the window.

The menu and the toolbar provide access to common editing functionalities such as
Load/Save, Cut/Copy/Paste, Redo/Undo and Zoom in/Zoom out.

When the circuit is ready to run, clickon the toolbar. This operation will activate
the DE2 board for simulation. To stop the simulation, simply click.

Finally, if a high speed implementation of the circuit is required, click  on the toolbar.
This will bring up a dialog asking for a folder to save the Verilog files. A self-contained project
will be saved into this folder, which is ready to be synthesised by Quartus.
